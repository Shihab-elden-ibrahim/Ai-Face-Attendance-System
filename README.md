# Ai-Face-Attendance-System
AI Face Recognition Attendance System
Overview

An AI-powered attendance system that uses face recognition to automatically detect and record attendance in real time. The project captures facial data through a webcam, trains a recognition model, and stores attendance records in an Excel sheet.

Features
Real-time face detection and recognition
Automatic attendance logging with date and time
Excel-based attendance storage
Dataset collection and model training
Duplicate attendance prevention
Technologies Used
Python
OpenCV
NumPy
OpenPyXL
How It Works
Register new faces using the webcam.
Train the face recognition model.
Start the recognition system.
Attendance is automatically saved when a registered face is detected.
